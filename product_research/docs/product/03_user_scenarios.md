## Пользовательские сценарии: Продажа РКО для ИП и малого бизнеса

### Сценарий 1: Менеджер проходит тренировку впервые по РКО

**Как** новый менеджер по работе с малым бизнесом,  
**я хочу** самостоятельно выбрать сценарий «РКО для ИП» и пройти тренировочный звонок,  
**чтобы** отработать навыки до первого реального звонка предпринимателю.

**Поток:**
1. Менеджер заходит в Личный кабинет.
2. Система предлагает выбрать продукт: «РКО для ИП/OOO».
3. Менеджер выбирает уровень сложности 1 (для новичков) и архетип клиента «Новичок» — дружелюбный ИП, только открыл магазин, не разбирается в комиссиях.
4. Нажимает «Начать тренировку».
5. Система запускает голосовой звонок: ИИ-клиент говорит: «Здравствуйте, у меня ИП, продаю цветы. Сейчас в Тинькофф, но комиссии высокие».
6. Менеджер выявляет потребность, представляет выгоды (бесплатно 3 месяца, эквайринг 0% первый месяц, онлайн-бухгалтерия).
7. После завершения звонка система показывает обратную связь с оценкой, и рекомендациями.
8. Менеджер может сразу пройти тренировку снова или выбрать другой архетип.

### Сценарий 2: Менеджер тренируется на скептически настроенном клиенте

**Как** менеджер, работающий с опытными предпринимателями,  
**я хочу** пройти тренировку с клиентом-скептиком, который не верит в выгоды,  
**чтобы** отработать аргументацию и расчёт ROI.

**Поток:**
1. Менеджер выбирает продукт «РКО для ИП», уровень сложности 3, архетип «Скептик».
2. ИИ-клиент говорит: «У меня всё устроено. Зачем менять банк? Все обещают “бесплатно”, а потом берут в 2 раза дороже».
3. Менеджер должен чётко объяснить:  
   - Точную стоимость после льготного периода  
   - Экономию по сравнению с текущим банком  
   - Наличие персонального менеджера
4. Получает обратную связь с акцентом на:  
   - Достоверность фактов (совпадают ли тарифы с базой знаний)  
   - Умение перевести разговор в расчёт выгоды
5. В рекомендации: «Попробуйте спросить: “Сколько вы сейчас платите за переводы и эквайринг?” — это основа для аргументации».

### Сценарий 3: Тренер анализирует прогресс по РКО

**Как** старший тренер отдела малого бизнеса,  
**я хочу** видеть аналитику по всем менеджерам, работающим с РКО,  
**чтобы** выявить, кто слабо аргументирует выгоды или не выявляет потребность.

**Поток:**
1. Тренер заходит в Личный кабинет тренера.
2. Фильтрует по продукту: «РКО для ИП/OOO».
3. Видит список менеджеров с колонками: кол-во тренировок, средний балл, оценка по «выявлению потребности».
4. Выбирает менеджера «Козлов Д.» — у него 8 тренировок, но балл по «работе с возражениями» = 0.8/2.
5. Открывает детальный отчёт:  
   - Часто пропускает вопрос «Какие комиссии вас не устраивают?»  
   - Не упоминает онлайн-бухгалтерию как выгоду
6. Назначает индивидуальную тренировку на тему «Возражение: “Мне не нужно менять банк”».

### Сценарий 4: Тренер сравнивает подходы к РКО

**Как** тренер,  
**я хочу** сравнить, как два менеджера продают РКО,  
**чтобы** выявить эффективные техники и масштабировать их в команде.

**Поток:**
1. Тренер выбирает двух менеджеров: «Ермакова» и «Волков».
2. Сравнивает их по сценарию «РКО для ИП».
3. Видит:  
   - Ермакова: всегда спрашивает про текущие расходы на РКО → сильная аргументация  
   - Волков: быстро переходит к презентации → клиент не чувствует персонализации
4. Делает вывод: «Техника “узнай текущую боль” повышает конверсию».
5. Включает пример звонка Ермаковой в обучающий материал для новых сотрудников.

### Сценарий 5: Методолог обновляет сценарий РКО после изменения условий

**Как** методолог,  
**я хочу** обновить сценарий «РКО для ИП/OOO» после запуска новой акции,  
**чтобы** менеджеры не обещали устаревшие условия.

**Поток:**
1. Методолог заходит в раздел «Сценарии».
2. Выбирает «РКО для ИП v1».
3. Меняет в базе знаний:  
   - Бесплатный период: с 1 до 3 месяцев  
   - Эквайринг: 0% — 1 месяц (было — 2 недели)
4. Обновляет обязательные фразы: добавляет «Вы получите онлайн-бухгалтерию в подарок на 6 месяцев».
5. Сохраняет как «РКО для ИП v2».
6. Все новые тренировки автоматически используют обновлённую версию.

### Сценарий 6: Менеджер пересдаёт тренировку по РКО после ошибки в compliance

**Как** менеджер,  
**я хочу** исправить ошибку, связанную с неправильным описанием условий РКО,  
**чтобы** не нарушать регуляторные требования и не вводить клиента в заблуждение.

**Поток:**
1. Менеджер проходит тренировку и получает оценку 4/10.
2. В обратной связи: «Вы сказали “РКО навсегда бесплатно” — это неверно. После льготного периода — 990 ₽/мес».
3. Нажимает «Повторить».
4. Во второй попытке чётко говорит: «Первые 3 месяца — бесплатно, далее — 990 ₽ в месяц».
5. Добавляет: «Вы можете отключить услугу в любой момент».
6. Получает 9/10 и разблокирует доступ к другим сценариям.
